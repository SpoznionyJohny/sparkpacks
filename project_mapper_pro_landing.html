FAQ

Questions.

What languages does it support?
Deep analysis for Python. Directory tree and structure for JavaScript, TypeScript, Go, Java, Ruby, PHP, Rust, C and C#.
Do I need to install anything?
No. It's a single Python file (~900 lines). Copy it to your project, run it. That's it. Python 3.8+ required — that's it.
How big can the project be?
Tested on projects with 200+ files. It handles large codebases cleanly — skips binary files, respects .gitignore patterns.
Does it work with monorepos?
Yes. Run it from the root and it maps everything. Run it from a subdirectory to map just that service.
